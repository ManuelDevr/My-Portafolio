# Landing Page Personal

Este proyecto es una landing page personal creada con React, TypeScript y Vite.

## Descripción

La landing page está pensada para mostrar una presentación personal clara y atractiva, con transiciones suaves, enlaces a redes sociales y con buen diseñoS.

## Tecnologías usadas

- React 19
- TypeScript
- Vite
- Tailwind CSS
